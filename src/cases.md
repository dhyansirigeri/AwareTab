This doc provides idea on how the various UI would change based on user interactions.

all themes = stressed, focused, tired, relaxed

1) Stressed: show stressed theme ui, with breathing ui (to be explained later).
components to display: music, weather, clock, guided breathing component, settings

2) Focused: show focused theme ui, load specific background.
components to display: clock, music, focus timer component, settings, bookmarks, google apps,
                        to do list component. 

3) Relaxed/Normal: show default ui and display all default components.

4) Tired: show dusky tired ui and background. 
components to display: same as normal mode, but with dusky theme.
